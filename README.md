# Brew & Butter

[Live website](https://brew-and-butter.vercel.app)

Multi-location café website for Brew & Butter in Beirut. The site gives customers fast paths to branch information, ordering, study-space details, and opening hours while keeping the brand visually distinctive.

![Brew and Butter website preview](https://raw.githubusercontent.com/hsen308/hsen308/main/assets/projects/brew-and-butter.webp)

## My role

I designed and built the experience end to end through Novu Studios, including information architecture, responsive implementation, deployment, and local-business metadata.

## Highlights

- Separate Downtown Beirut and Mousaytbe location information
- WhatsApp and delivery ordering paths
- Responsive layouts for mobile and desktop
- Open Graph metadata and `CafeOrCoffeeShop` structured data
- Performance-focused single-page implementation

## Stack

HTML · CSS · JavaScript · JSON-LD · Vercel

## Run locally

Open `index.html` directly or serve the folder with any static web server:

```bash
python -m http.server 8000
```
